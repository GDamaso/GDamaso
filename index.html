About

Information and Computer Systems

I am a Linux enthusiast and software developer who recently graduated from the Information and Computer Systems program at Camosun College in Victoria. During my studies, I developed various projects that honed my skills across multiple domains. I created a Python server running a simplified CLI version of Civilization, controlled through TCP packets, then reimplemented it in django. Additionally, I built a 3 Wordle-style web application using AWS infrastructure: Lambda, RDS, and API Gateway. I also developed a slow-motion, fps, platform archery game in Unity. For my capstone project, my team, QuackStack, was selected by the BC Public Service Sustainment Team to develop a prototype to replace a nutrient management calculator used by thousands of BC farmers.
